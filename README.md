## Clone the repo

``` bash
git clone https://github.com/Inert-one/video-streaming-platform.git
```

## Setup and Start Node Server
``` bash
cd server
yarn install
yarn dev
```

## Setup and Start React Application
``` bash
cd client
yarn install
yarn start
```
